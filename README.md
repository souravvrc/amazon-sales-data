# 📊 Amazon Sales Analysis 

## 📌 Project Overview

This project focuses on analyzing Amazon Sales Data to uncover business insights, identify sales trends, and evaluate key performance metrics using Data Analytics and Business Intelligence techniques.

The analysis includes data cleaning, preprocessing, exploratory data analysis (EDA), visualization, and dashboard creation to help understand sales performance across different regions, countries, item types, and sales channels.

---

## 🎯 Objectives

- Perform ETL (Extract, Transform, Load) operations on Amazon Sales data.
- Analyze sales trends:
  - Month-wise Sales
  - Year-wise Sales
  - Year-Month Sales Patterns
- Identify key business metrics.
- Discover relationships between sales attributes.
- Build interactive dashboards for decision-making.

## 📈 Exploratory Data Analysis (EDA)

### Key Business Questions Solved

1. Which region has the highest total sales revenue?
2. Which country generates the highest profit?
3. What is the average unit price and unit cost for each item type?
4. How does sales channel affect order priority distribution?
5. What is the average order processing time?
6. Which item type generates the highest and lowest revenue?
7. How does order priority vary across regions?
8. Is there a relationship between unit price and profit?
9. Are there seasonal trends in sales?
10. How do units sold vary across countries?
11. Which sales channel has the highest average unit price?
12. Are there outliers in total cost distribution?
13. How does profit vary across item categories?

---

## 📊 Key Findings

### 🌍 Highest Revenue Region
- Sub-Saharan Africa generated the highest total revenue.

### 💰 Highest Profit Country
- Djibouti recorded the highest total profit.

### 🏆 Best Performing Product Category
- Cosmetics generated the highest revenue and profit.

### 📉 Lowest Performing Product Category
- Fruits generated the lowest revenue and profit.

### 🔗 Correlation Analysis
- Strong positive correlation observed between:
  - Unit Price
  - Total Profit

### ⏱️ Processing Time
- Average processing time was approximately 23 days for both Online and Offline sales channels.

### 📅 Seasonal Insights
- Certain months showed significantly higher revenue, indicating seasonal demand patterns.
